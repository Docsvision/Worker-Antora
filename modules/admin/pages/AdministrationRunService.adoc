= Запуск Службы рабочих процессов

Работу Модуля обеспечивает служба «Docsvision 5.5 Worker Service». Данная служба должна быть запущена.

Для корректной работы службы рабочих процессов с использованием сервиса перештамповки подписей необходимо использовать машину, имеющую как минимум два процессора.
